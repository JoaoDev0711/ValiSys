Notas de atualização

Principais mudanças aplicadas nesta versão do ValiSys.

Versão minimalista profissional

Interface simplificada

Visual mais limpo, telas mais centradas, menos elementos visuais e melhor leitura para operação em celular.

Acesso

Código de acesso visível no login

A tela de login agora possui botão para mostrar ou ocultar o código digitado antes de entrar.

Orientação

Tutorial no primeiro acesso

O sistema apresenta um guia rápido no primeiro login e mantém uma página de tutorial completa.

Operação

Ações rápidas em páginas internas

As principais telas receberam atalhos para Início, Tutorial, Novidades, Atualizar e Sair.

Financeiro

Planos e Minha assinatura

Foram adicionadas páginas internas de planos, assinatura, cobranças dos próximos 12 meses e meios de pagamento configuráveis.
