ValiSys - Fluxo correto de acesso

1. Funcionário comum
- Abre o sistema.
- Escolhe a loja.
- Vai para a tela de entrada.
- Seleciona o tipo de acesso.
- Promotor entra com nome.
- Encarregado e gerente entram com nome + código cadastrado.

2. Admin geral
- Entra pela tela de entrada como Admin geral.
- Usa a senha admin123.
- Depois escolhe a loja que quer administrar.

3. Cadastro de funcionários
- Gerente e admin podem cadastrar funcionários.
- Cada funcionário fica ligado à loja atual.
- Cada funcionário pode ter setor.
- Gerente e encarregado usam código de acesso.
- Promotor pode entrar de forma simples pela loja.

4. Permissões
- Admin: administra tudo.
- Gerente: vê a loja toda.
- Encarregado: quando tiver setor cadastrado, vê o setor dele.
- Promotor: lança produtos e vê seus próprios lançamentos.
