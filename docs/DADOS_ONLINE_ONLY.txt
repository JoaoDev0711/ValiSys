ValiSys sistema-only

Nesta versão, os dados principais não são salvos no celular.

sistema:
- lojas
- funcionarios
- produtos
- lancamentos
- notificacoes

localStorage fica somente para sessão:
- usuarioLogado
- lojaAtual

Isso é necessário para manter o usuário e a loja selecionada enquanto navega entre páginas.

Para testar:
1. Confirme que database/dados_online-setup.sql já foi rodado.
2. Confirme que js/dados_online-config.js tem a URL e a publishable key.
3. Abra escolher-loja.html.
4. Crie/seleciona loja.
5. Lance um produto.
6. Abra em outro celular com o mesmo link.
7. Entre e escolha a mesma loja.
8. A lista geral deve puxar os dados do sistema.
