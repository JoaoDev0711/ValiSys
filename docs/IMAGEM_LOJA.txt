Imagem da loja

Agora gerente e admin podem cadastrar uma imagem para a loja usando arquivo do celular.

Como funciona:
- Ao cadastrar loja, escolha uma imagem no campo "Imagem da loja".
- A imagem é compactada no navegador antes de salvar.
- Nas lojas já cadastradas, use "Trocar imagem".
- Se a loja não tiver imagem, o sistema mostra as iniciais do nome.

Exemplo:
Mercado Demonstração = MD
Loja São João = LS
Atacadão Central = AC

Antes de usar em uma base já criada, rode:
database/valisys-sql-unico.sql
